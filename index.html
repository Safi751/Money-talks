🎮 1. Start the Game Install and open GTA San Andreas. Click Start Game. Choose New Game or Load Game. You play as Carl Johnson (CJ). 🕹 2. Basic Controls (PC) Movement W = Forward S = Back A = Left D = Right Shift = Run Space = Jump Fighting Left Mouse = Punch / Shoot Right Mouse = Aim R = Reload Driving F or Enter = Enter/Exit vehicle W = Accelerate S = Brake/Reverse A / D = Turn 🗺 3. Follow Missions Look at the map (press M). Go to the mission marker (shown as letters like CJ, BS, etc.). Stand in the red circle to start mission. Follow instructions on screen. 💰 4. Earn Money Complete missions. Do side jobs (Taxi, Police, Ambulance). Win races. Buy properties. 🔫 5. Improve Skills Go to gym to increase strength. Drive more to improve driving skill. Shoot more to improve weapon accuracy. ❤️ 6. Health & Armor Eat food to restore health. Wear armor for extra protection. Avoid police stars ⭐ (higher stars = more police).
